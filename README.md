# 📝 Todo List App

A modern, responsive, and interactive Todo List web application built using **HTML, CSS, and Vanilla JavaScript**.

This project helps users manage daily tasks efficiently with a clean UI, smooth interactions, and persistent storage using Local Storage.

## ✨ Features

### ✅ Task Management
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Undo completed tasks
- Prevent empty tasks

### 💾 Data Persistence
- Saves tasks in Local Storage
- Automatically loads tasks on refresh

### 🎨 UI & UX
- Clean and minimal design
- Responsive layout (mobile + desktop)
- Smooth hover effects
- Simple and user-friendly interface

### ⚡ Interaction Features
- Instant task updates
- Keyboard support (Enter key to add tasks)
- Real-time UI updates without reload

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Local Storage API
